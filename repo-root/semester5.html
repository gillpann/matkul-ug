Pemrograman WEB

  • Pengenalan Website

    Definisi Website: Website adalah kumpulan halaman web yang dapat diakses melalui internet, biasanya dihosting di server web, dan diidentifikasi dengan nama domain. Website bisa statis atau dinamis.

    Sejarah dan Evolusi: Website pertama kali diperkenalkan oleh Tim Berners-Lee pada tahun 1991. Sejak itu, teknologi web berkembang pesat, dari HTML dasar menjadi HTML5, dan dari situs web statis menjadi aplikasi web interaktif dan responsif.

  • Unsur-unsur Website

    1. Struktur Dasar Website: Website dibangun menggunakan HTML (Hypertext Markup Language) yang menyediakan struktur dasar halaman web, seperti paragraf, tautan, gambar, dan formulir.

    2. Styling (CSS): Cascading Style Sheets (CSS) digunakan untuk mengatur tata letak dan penampilan halaman web, termasuk warna, font, dan tata letak responsif untuk berbagai perangkat.

    3. Interaktivitas (JavaScript): JavaScript adalah bahasa pemrograman yang memungkinkan pengembang untuk membuat halaman web interaktif, seperti validasi formulir, animasi, dan manipulasi elemen DOM (Document Object Model).

  • Fungsi/Manfaat Website

    1. Komunikasi: Website memungkinkan komunikasi efektif antara individu dan organisasi melalui blog, forum, dan media sosial.

    2. E-commerce: Platform seperti Shopee dan Tokopedia memungkinkan penjualan dan pembelian produk secara online, menawarkan transaksi yang aman dan beragam pilihan produk.

    3. Branding: Website dapat memperkuat citra merek dengan desain yang konsisten dan konten yang relevan, membantu membangun hubungan dengan pelanggan.

    4. Komunitas: Website komunitas atau forum memungkinkan pengguna untuk berbagi pengetahuan, ide, dan dukungan, seperti Stack Overflow untuk pengembang perangkat lunak

Jenis Pemrograman WEB

Client-Side

Pemrograman Sisi Klien (Client-Side): Pemrograman ini dilakukan pada browser pengguna dan berfokus pada antarmuka pengguna. Teknologi utamanya adalah HTML, CSS, dan JavaScript. Framework seperti React atau Vue.js sering digunakan untuk membangun aplikasi web yang kompleks dan dinamis.

Server-Side

Pemrograman Sisi Server (Server-Side): Pemrograman ini berjalan di server dan menangani logika aplikasi, penyimpanan data, dan autentikasi. Contohnya adalah PHP, Python (dengan Django atau Flask), dan Node.js. Server-side scripting mengelola komunikasi antara server dan browser.

Full-Stack

Full-Stack Developer memiliki keahlian dalam pengembangan sisi klien dan sisi server. Mereka dapat bekerja di seluruh bagian aplikasi, dari database hingga antarmuka pengguna. Contoh tumpukan teknologi full-stack termasuk MERN (MongoDB, Express.js, React, Node.js).

- Konsep dalam Pembuatan Website-

Frontend

Melibatkan segala sesuatu yang berhubungan dengan tampilan dan interaksi pengguna di browser. Teknologi frontend mencakup HTML, CSS, JavaScript, dan framework seperti Angular, Vue.js, atau React. Tugas frontend meliputi desain UI (User Interface) dan UX (User Experience).

Backend

Berfokus pada logika bisnis, manajemen basis data, dan fungsionalitas server. Bahasa backend seperti Node.js, Ruby on Rails, dan Java dengan Spring Boot digunakan untuk mengelola data dan merespons permintaan klien.

Fullstack

Menggabungkan keahlian frontend dan backend, memungkinkan pengembang untuk membangun dan mengelola seluruh aplikasi dari ujung ke ujung. Full-Stack Developer bekerja pada semua bagian aplikasi, dari pengembangan API hingga integrasi UI.
